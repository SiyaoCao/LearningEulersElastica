## 2. 辅助激活函数类 `act`（第 9–13 行）

```python
9.  class act(nn.Module):
10.   def __init__(self,):
11.     super().__init__()
12.   def forward(self,x):
13.     return torch.sigmoid(x)*x
```

**说明：**

该类封装了一种被称为 **Swish** 的激活函数，在 `theta_net` 的 DeepONet 分支中使用（无论 `act_name` 取何值，这里的 `act()` 均指代 Swish）。

**第 13 行**对应的数学定义：

\[
\text{Swish}(x) = \sigma(x) \cdot x = \frac{x}{1 + e^{-x}}
\]

其中 \(\sigma(x) = \frac{1}{1+e^{-x}}\) 是 sigmoid 函数。与 ReLU 相比，Swish 是可微的且非单调的，有助于梯度的顺畅流动。

## 3. 参数化子网络类 `theta_net`（第 15–94 行）

`theta_net` 是整个角度神经网络的核心参数化部分，记为 \(\hat{\theta}_{\rho}^{c}\)。它接收弧长坐标 \(s\) 和边界处切向量角度 \(\theta_0, \theta_N\) 作为输入，输出角度的预测值（单位为弧度）。

### 3.1 `__init__` 方法（第 16–51 行）

```python

18.
```

**逐行说明：**

- **第 16 行**：构造函数接受以下参数：
  - `act_name`：激活函数名称，可选 `'tanh'`、`'sin'`、`'swish'` 或其他（默认为 sigmoid）。
  - `nlayers`：隐藏层数量。
  - `hidden_nodes`：每层的神经元数量。
  - `is_res`：布尔值，是否使用残差连接（ResNet 风格）。
  - `is_deeponet`：布尔值，是否采用 DeepONet 架构。

- **第 19–20 行**：固定随机种子，保证实验可复现性。

- **第 22–29 行**：根据 `act_name` 设置激活函数 \(\sigma(\cdot)\)。四种选择：

  \[
  \sigma(x) = \begin{cases}
  \tanh(x) & \text{if act\_name} = \texttt{'tanh'} \\
  \sin(x)  & \text{if act\_name} = \texttt{'sin'} \\
  x \cdot \text{sigmoid}(x) & \text{if act\_name} = \texttt{'swish'} \\
  \text{sigmoid}(x) & \text{otherwise}
  \end{cases}
  \]

- **第 36 行**：定义嵌入层 `self.embed`，将三维输入（\(\theta_1/\pi,\, \theta_2/\pi,\, s\)）映射到隐藏空间：

  \[
  \mathbf{h}_{\text{embed}} = W_{\text{embed}} \begin{pmatrix} \theta_1/\pi \\ \theta_2/\pi \\ s \end{pmatrix} + \mathbf{b}_{\text{embed}}, \quad W_{\text{embed}} \in \mathbb{R}^{d \times 3}
  \]

  其中 \(d\) 为 `hidden_nodes`。

- **第 39–42 行**（DeepONet 分支所需层）：`lift_U`、`lift_V`、`lift_H` 是 DeepONet trunk 网络中使用的三个辅助线性层，形状均为 \(\mathbb{R}^{d \times d}\)，配合 Hadamard 乘积实现乘法交互（MULT 架构）。

- **第 44 行**：`linears_Z` 是 DeepONet 中每层的线性变换列表，长度为 `nlayers`。

- **第 46 行**：`self.lift` 是标准 MLP/残差分支的提升层。

- **第 48–49 行**：`linears` 和 `linearsO` 分别是标准分支中每层的内层线性变换和外层线性变换，长度均为 `nlayers`。

- **第 51 行**：`self.proj` 是最终投影层，将隐藏向量映射为标量输出（即角度 \(\hat{\theta}\)）：

  \[
  \hat{\theta}_{\text{raw}} = W_{\text{proj}} \mathbf{h} + b_{\text{proj}}, \quad W_{\text{proj}} \in \mathbb{R}^{1 \times d}
  \]

### 3.2 `find_theta` 辅助方法（第 53–54 行）

```python
53.   def find_theta(self,v):
54.     return torch.atan2(v[:,1:2], v[:,0:1])
```

**说明：**

该方法计算切向量 \(\mathbf{v} = (v_x, v_y)\) 与 \(x\) 轴之间的夹角（弧度）：

\[
\theta = \arctan2(v_y,\, v_x), \quad \theta \in (-\pi, \pi]
\]

`torch.atan2(y, x)` 是四象限反正切函数，能正确判断向量所在的象限。`v[:,1:2]` 提取所有样本的 \(v_y\) 分量，`v[:,0:1]` 提取 \(v_x\) 分量，返回形状为 \((B, 1)\) 的张量（\(B\) 为 batch size）。

### 3.3 `forward` 前向传播方法（第 57–94 行）

```python
57.   def forward(self,s,q1,q2,v1,v2):
58.     s = s.reshape(-1,1)
59.     q1 = q1.reshape(-1,2)
60.     q2 = q2.reshape(-1,2)
61.     v1 = v1.reshape(-1,2)
62.     v2 = v2.reshape(-1,2)
63.
64.     theta1 = self.find_theta(v1) / torch.pi
65.     theta2 = self.find_theta(v2) / torch.pi
66.
67.     #input = torch.cat((theta1,theta2,q1,q2,s),dim=1)
68.     input = torch.cat((theta1,theta2,s),dim=1)
69.     input = torch.sin(2*torch.pi*self.embed(input))
70.
71.     if self.is_deeponet:
72.         U = self.act(self.lift_U(input))
73.         V = self.act(self.lift_V(input))
74.
75.         H = self.act(self.lift_H(input))
76.
77.         for i in range(self.nlayers):
78.             Z = self.linearsO[i](self.act(self.linears_Z[i](H)))
79.             H = U*(1-Z) + V*Z
80.
81.         input = H
82.
83.     else:
84.
85.         input = self.act(self.lift(input))
86.
87.         for i in range(self.nlayers):
88.           if self.is_res:
89.               input = input + self.linearsO[i](self.act(self.linears[i](input)))
90.           else:
91.               input = self.act(self.linears[i](input))
92.
93.     output = self.proj(input)
94.     return output*torch.pi
```

**逐行说明：**

**第 58–62 行 — 输入整形：**

将所有输入调整为标准形状：
- \(s\) 调整为 \((B, 1)\)（弧长坐标，批量大小为 \(B\)）
- \(\mathbf{q}_1, \mathbf{q}_2\) 调整为 \((B, 2)\)（左/右端点位置）
- \(\mathbf{v}_1, \mathbf{v}_2\) 调整为 \((B, 2)\)（左/右端点切向量）

**第 64–65 行 — 提取边界角度：**

\[
\theta_1 = \frac{1}{\pi}\arctan2\!\left(v_{1,y},\, v_{1,x}\right), \qquad
\theta_2 = \frac{1}{\pi}\arctan2\!\left(v_{2,y},\, v_{2,x}\right)
\]

将角度归一化到 \([-1, 1]\) 区间，使网络输入处于合理的数值范围内。

**第 68 行 — 构建网络输入向量：**

将归一化的边界角度和弧长坐标拼接：

\[
\mathbf{z} = \begin{pmatrix} \theta_1/\pi \\ \theta_2/\pi \\ s \end{pmatrix} \in \mathbb{R}^3
\]

（注释掉的第 67 行曾包含端点位置 \(\mathbf{q}_1, \mathbf{q}_2\)，但当前版本仅使用角度和 \(s\)。）

**第 69 行 — 正弦嵌入（Fourier 特征）：**

\[
\mathbf{h}^{(0)} = \sin\!\left(2\pi \cdot \left(W_{\text{embed}}\,\mathbf{z} + \mathbf{b}_{\text{embed}}\right)\right) \in \mathbb{R}^d
\]

这一步实现了 **随机 Fourier 特征映射**（类似 NeRF 中的位置编码），将低维输入映射到高维正弦特征空间，有助于网络学习高频函数。

---

**DeepONet 分支（第 71–81 行），当 `is_deeponet=True` 时执行：**

**第 72–75 行 — 初始化三条辅助流：**

\[
\mathbf{U}^{(0)} = \sigma\!\left(W_U\,\mathbf{h}^{(0)} + \mathbf{b}_U\right), \quad
\mathbf{V}^{(0)} = \sigma\!\left(W_V\,\mathbf{h}^{(0)} + \mathbf{b}_V\right), \quad
\mathbf{H}^{(0)} = \sigma\!\left(W_H\,\mathbf{h}^{(0)} + \mathbf{b}_H\right)
\]

**第 77–80 行 — 多层 MULT（乘法交互）传播：**

对第 \(l = 0, 1, \ldots, L-1\) 层重复：

\[
\mathbf{Z}^{(l)} = W_{O,l}\,\sigma\!\left(W_{Z,l}\,\mathbf{H}^{(l)} + \mathbf{b}_{Z,l}\right) + \mathbf{b}_{O,l}
\]

\[
\mathbf{H}^{(l+1)} = \mathbf{U}^{(0)} \odot \left(\mathbf{1} - \mathbf{Z}^{(l)}\right) + \mathbf{V}^{(0)} \odot \mathbf{Z}^{(l)}
\]

其中 \(\odot\) 表示 Hadamard（逐元素）乘积。此 MULT 架构通过门控机制让网络在 \(\mathbf{U}\) 和 \(\mathbf{V}\) 两个方向之间进行可学习的插值，增强了表达能力。

---

**标准 MLP/残差分支（第 83–91 行），当 `is_deeponet=False` 时执行：**

**第 85 行 — 提升：**

\[
\mathbf{h}^{(0)} = \sigma\!\left(W_{\text{lift}}\,\mathbf{h}^{(0)} + \mathbf{b}_{\text{lift}}\right)
\]

**第 87–91 行 — 多层前向传播：**

若 `is_res=True`（残差网络）：

\[
\mathbf{h}^{(l+1)} = \mathbf{h}^{(l)} + W_{O,l}\,\sigma\!\left(W_{l}\,\mathbf{h}^{(l)} + \mathbf{b}_{l}\right) + \mathbf{b}_{O,l}
\]

若 `is_res=False`（标准 MLP）：

\[
\mathbf{h}^{(l+1)} = \sigma\!\left(W_{l}\,\mathbf{h}^{(l)} + \mathbf{b}_{l}\right)
\]

---

**第 93–94 行 — 最终输出（投影与反归一化）：**

\[
\hat{\theta}_{\rho}^{c}(s, \theta_0, \theta_N) = \pi \cdot \left(W_{\text{proj}}\,\mathbf{h}^{(L)} + b_{\text{proj}}\right)
\]

将最终隐藏特征通过线性层映射为标量，再乘以 \(\pi\) 将输出反归一化回弧度范围 \(\approx [-\pi, \pi]\)。这与第 64–65 行的归一化操作对称。

### 4.3 `local_poly_left` 方法（第 110–111 行）

```python
110.   def local_poly_left(self,s,d=1/30):
111.     return (d - s)**3/d**3
```

**说明：**

该方法定义一个局部多项式，用于在左边界附近平滑地施加边界条件（此方法在当前 `theta` 实现中未被直接调用，是历史遗留版本的辅助函数）。

其数学形式为：

\[
p_{\text{left}}(s;\, d) = \left(\frac{d - s}{d}\right)^3, \quad s \in [0, d]
\]

该多项式满足以下性质：
- \(p_{\text{left}}(0) = 1\)（在左端点取值为 1）
- \(p_{\text{left}}(d) = 0\)，\(p_{\text{left}}'(d) = 0\)，\(p_{\text{left}}''(d) = 0\)（在 \(s=d\) 处与其前两阶导数均消失）

默认参数 \(d = 1/30\)，即局部支撑区间宽度为 \(L/30\) 的比例。

### 4.4 `local_poly_right` 方法（第 112–113 行）

```python
112.   def local_poly_right(self,s,d=1/30):
113.     return (d + s - L)**3/d**3
```

**说明：**

与 `local_poly_left` 对称，用于右边界附近的平滑过渡（同样为历史遗留辅助函数）：

\[
p_{\text{right}}(s;\, d) = \left(\frac{(s - L) + d}{d}\right)^3, \quad s \in [L-d, L]
\]

满足：
- \(p_{\text{right}}(L) = 1\)（在右端点取值为 1）
- \(p_{\text{right}}(L-d) = 0\)，且其前两阶导数在 \(s = L - d\) 处消失

### 4.5 `theta` 方法——构建满足边界条件的角度函数（第 115–138 行）

```python
115.   def theta(self,s,q1,q2,v1,v2):
116.     L = 3.3
117.     s = s.reshape(-1,1)
118.     q1 = q1.reshape(-1,2)
119.     q2 = q2.reshape(-1,2)
120.     v1 = v1.reshape(-1,2)
121.     v2 = v2.reshape(-1,2)
122.
123.     parametric_part = self.parametric_part(s,q1,q2,v1,v2)
124.
125.     if self.impose_bcs:
126.       left_node = torch.zeros_like(s).to(s.device)
127.       right_node = (torch.ones_like(s)*self.L).to(s.device)
128.
129.       theta_1 = self.find_theta(v1)
130.       theta_2 = self.find_theta(v2)
131.
132.
133.       g_left = self.parametric_part(left_node,q1,q2,v1,v2).to(s.device)
134.       g_right =  self.parametric_part(right_node,q1,q2,v1,v2).to(s.device)
135.
136.       return  parametric_part + (theta_1-g_left) * torch.exp(-100*s**2) + (theta_2-g_right) * torch.exp(-100*(s-self.L)**2)
137.     else:
138.       return parametric_part
```

**逐行说明：**

这是整个角度网络的核心方法，实现论文中的公式 (17) 和 (18)。

**第 116–121 行**：局部变量 \(L = 3.3\)，以及对所有输入张量的整形处理。

**第 123 行**：调用 `theta_net` 得到参数化部分（即公式 (17)）：

\[
f_{\rho}(s,\, \theta_0,\, \theta_N) = \hat{\theta}_{\rho}^{c}(s,\, \theta_0,\, \theta_N)
\]

**第 125–136 行（`impose_bcs=True`，对应公式 (18)）：**

**第 126–127 行**：构造左边界坐标 \(s=0\) 和右边界坐标 \(s=L\) 的张量（形状与 \(s\) 相同，用于批量评估边界处的网络输出）。

**第 129–130 行**：计算真实的边界角度：
